# Mocko skill

A skill for writing and reviewing [Mocko](https://github.com/mocko-app/mocko) mocks. Covers mock files structure, the Bigodon templating language used in mock bodies, request context, data blocks, host blocks, flags, proxying and common patterns.

Once added to any skills-compatible agent, it can author new mocks, debug failing templates, and explain existing ones. This skill also helps with migration from v1 to v2.

## Install

```sh
npx skills@latest add mocko-app/skills/mocko
```
